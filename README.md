# Tuffy Reviews

Browser extension for Amazon, Walmart, Target, and Best Buy: aggregated reviews, price history, seller comparison, and trust scores on product pages.

## Run in Chrome

1. Open `chrome://extensions`, turn on **Developer mode**.
2. Click **Load unpacked** and select this folder (the one with `manifest.json`).
3. Open a product page on a supported site and click the extension icon.
